# Inbox

This folder is for quick notes, ideas, and temporary items.

## Workflow

1. **Capture**: Write quick notes here
2. **Process**: Review regularly (daily/weekly)
3. **Organize**: Move to appropriate folders
   - Planning → `01-Planning/`
   - Ideas → `04-Notes/Ideas.md`
   - Architecture → `02-Architecture/`
   - etc.

## Tips

- Don't let items sit here too long
- Process inbox regularly
- Link to related notes when moving items
