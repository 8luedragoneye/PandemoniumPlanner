# Transport System Overview

## Purpose

The transport system manages organized transport runs from Lymhurst to Carleon and back for the Albion Online alliance Schattenwandler.

## Key Concepts

### Definitions
- **Hinweg**: Lymhurst → Carleon
- **Rückweg**: Carleon → Lymhurst
- **Lauf**: Hinweg + Rückweg
- **Session**: Transport event consisting of multiple runs over 2h
- **Fighter**: Member of the fight party
- **Transporter**: Member of the transport party
- **Paar-Inventar**: Combined transport inventory of a Fighter/Transporter pair

### Core Features
- **Pairing System**: Fighter/Transporter pairs for efficient transport
- **Fill System**: Weight and slot optimization
- **Fill Priority**: Point-based system for fill provider assignment
- **Gear Loaning**: Equipment lending system
